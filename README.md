# Notes App - REST API

A RESTful API for managing notes with Node.js, Express, and MongoDB.

## Features

- CRUD Operations
- Route Parameters
- Query Parameters
- Pagination
- Sorting

## Tech Stack

- Node.js
- Express.js
- MongoDB (Mongoose)

## Quick Start

```bash
npm install
npm run dev
```

## Environment Variables

Create `.env` file:
```
MONGO_URI=your_mongodb_uri
PORT=5000
```
